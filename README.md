# 5 hours Limit Notifier
Web app sends desktop notifications every 5 hours

# 概要
設定した時刻から5時間ごとにデスクトップ通知を送るアプリです。
次に通知がくるまであと何分あるか、残り時間もわかるようになっています。
5時間ごとにリフレッシュされるものを忘れないように作りました。

# 使い方
下記URLから誰でも使えます。
https://ykr2355.github.io/5h-Limit-Notifier/

1. ブラウザでデスクトップ通知を許可
2. 鉛筆マークから次回の通知日時を設定
3. アプリのタブを開きっぱなしにしておく
4. 設定日時にデスクトップ通知が届き、自動で5時間後に再設定される

## ローカル環境で動かしたい人向け
1. すべてのファイルを同一フォルダに置く
2. start_timer.batを実行する（localhostが立ち上がりブラウザでアプリが開きます）
3. 以下同上

# Overview
This app sends desktop notifications every 5 hours starting from a set time.
It also shows how many minutes remain until the next notification.
I made it to help you remember things that need refreshing every 5 hours.

# How to Use
Anyone can use it from the URL below.
https://ykr2355.github.io/5h-Limit-Notifier/

1. Allow desktop notifications in your browser
2. Set the next notification date/time via the pencil icon
3. Keep the app tab open
4. A desktop notification will arrive at the set time and automatically reschedule itself 5 hours later

## For those wanting to run it locally
1. Place all files in the same folder
2. Run start_timer.bat (localhost will launch and open the app in your browser)
3. Follow the steps above
